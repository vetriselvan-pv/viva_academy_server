Viva Academy Management System (VAMS)
Business Overview

Viva Academy is a coaching institution that operates multiple branches across different cities. Each branch offers multiple courses, employs faculty members, enrolls students, collects fees, schedules classes, conducts examinations, and tracks student progress.

Currently, most operations are handled manually using spreadsheets and paper records, leading to:

Duplicate student records
Fee collection errors
Poor attendance tracking
Lack of centralized reporting
Difficulty monitoring branch performance
No proper communication between branches
No audit trail

The objective is to build a centralized web application that allows management to control every branch from a single platform.
